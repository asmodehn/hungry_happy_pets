Develop
=======

Retrieve the source:
-------------------
git clone


First Timer (only once)
-----------
flask db init


Everyday Dev
------------

flask run

Testing
-------

pytest


Deploy - Staging
================

Heroku Local
------------
heroku local

Deploy - Production
===================

Heroku First Timer (only once)
------------------
heroku create

Heroku Deploy
-------------
heroku deploy
